# Invitation Sites

Проект для создания красивых пригласительных сайтов для свадеб, никаха, дней рождения, baby shower и других мероприятий.

## Идея

```text
готовые шаблоны → клиент выбирает → меняем данные → публикуем сайт → клиент получает ссылку
```

## MVP

- 1 каталог шаблонов
- 5 первых шаблонов
- 1 пример клиентского сайта
- документы: цены, бриф, процесс заказа, Instagram и лидогенерация

## Структура

```text
catalog/      каталог шаблонов
clients/      сайты клиентов
templates/    оригинальные шаблоны
docs/         бизнес-документы и процесс
assets/       общие картинки, шрифты, иконки, музыка
research/     референсы и дизайн-паттерны
```

## Правило

Чужие сайты используем только как референсы. Не копируем чужой код, фото, тексты, брендинг или дизайн 1-в-1.
