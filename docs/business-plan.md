# Business Plan

## Цель

Создать мини-фабрику пригласительных сайтов: быстро делать красивые сайты на базе оригинальных шаблонов.

## Клиенты

- пары на свадьбу;
- никах;
- дни рождения;
- baby shower;
- event-организаторы;
- фотографы и декораторы как партнёры;
- банкетные залы.

## MVP

```text
1 каталог + 5 шаблонов + 1 пример клиентского сайта
```

## Продажа

Instagram/WhatsApp → каталог → выбор шаблона → бриф → preview → правки → публикация.
