# Reference analysis: PRIGLASI_AVA catalog

Source catalog: https://priglasiabakirova.tilda.ws/webcatalogpriglasiava

Date: 2026-07-09

## Important rule

This catalog is used only as a market/reference source.

We must not copy:
- their code;
- their exact layouts 1:1;
- their texts;
- their images/photos;
- their icons/illustrations;
- their brand/name/contacts;
- their Tilda blocks as-is.

We can use:
- general structure ideas;
- user journey ideas;
- block types;
- pricing/FAQ ideas;
- category ideas;
- UX patterns;
- cultural/event-specific requirements.

## Catalog observations

The catalog contains many invitation website examples with repeated CTA buttons:

- `ПОСМОТРЕТЬ` — opens a demo/client invitation page.
- `ЗАКАЗАТЬ` — WhatsApp order CTA.
- Variants are named like `вариант № 1`, `вариант № 2`, etc.
- The catalog covers weddings, кыз узатуу, birthdays/jubilees, and other event formats.
- There is a FAQ/pricing section explaining what is included, paid changes, timing, payment, and site duration.

Useful catalog ideas for us:

1. Separate templates by event type.
2. Show many template cards with direct preview and order buttons.
3. Add WhatsApp CTA under each template.
4. Add FAQ below the catalog.
5. Add clear explanation of what can be changed for free and what costs extra.
6. Add package pricing and production time.
7. Add “custom design by request” option.

## Block library we should build

For our own templates, we should create reusable sections:

1. Hero
   - names / event title;
   - date;
   - photo or abstract background;
   - optional language switch.

2. Music
   - play/pause button;
   - default off until user taps.

3. Invitation text
   - wedding version;
   - nikah version;
   - kyz uzatuu version;
   - birthday/jubilee version;
   - baby shower version.

4. Calendar
   - highlighted event date;
   - month label;
   - responsive mobile design.

5. Schedule/timeline
   - 2–5 timeline items;
   - icons optional;
   - examples: guest gathering, ceremony, banquet, cake, end.

6. Venue/map
   - address;
   - 2GIS link;
   - Google Maps/Yandex Maps optional;
   - venue Instagram optional.

7. Dress code
   - color circles/swatches;
   - short client text;
   - optional example outfits.

8. Gallery
   - couple photos;
   - family photos;
   - event mood photos.

9. RSVP
   - guest name;
   - attendance yes/no/later;
   - number of guests;
   - comment field optional;
   - deadline text.

10. Countdown
    - days/hours/minutes/seconds;
    - shown near schedule or before RSVP.

11. After-event gallery link
    - Telegram/Google Drive link for guests to upload/view photos.

12. Footer CTA
    - “Want a site like this?”;
    - WhatsApp/Instagram links;
    - our own brand.
