Owner access

Вход владельца

Введите owner token один раз на своём устройстве. После сохранения админка сможет загружать заявки через защищённую Supabase Edge Function.

Token хранится только в localStorage этого браузера. Service-role key сюда не вставляется.

Безопасно: в браузер вводится только OWNER_API_TOKEN. SUPABASE_SERVICE_ROLE_KEY остаётся только в Supabase Edge Function.
